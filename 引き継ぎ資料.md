## 9. 注意事項・断念済み事項

### Accessフォームのデザイン変更
- **CreateControl で再作成する方式が安全**
- COMからデザインを変更する場合、直接プロパティ変更より一度DeleteControl→CreateControlの方が安定

### WebBrowserコントロール方式（断念済み）
- AccessのセキュリティダイアログでHTMLコンテンツがブロックされる
- `setup_webbrowser_ui.py` は参考用として保存

### EXE化（断念済み）
- Access ODBC の 32/64bit 問題で断念
- 現在のAccess FE/BE構成がベスト

### PDF出力（未実装）
- F_Report の `btnPDF_Click` は現在「開発中」メッセージを表示するのみ
- 実装候補: Access レポートオブジェクト → PDF出力 (`DoCmd.OutputTo acOutputReport, ..., acFormatPDF, ...)`)
- 参考: Flask版 `salesmgr_web/app.py` に reportlab 実装あり（15ページ、表紙+KPIカード+推移+ランキング+時間帯+担当者別）

## 12. 業務用語

| 用語 | 意味 |
|------|------|
| 架電 (calls) | 電話をかけること |
| 有効 (valid) | 相手が出た架電 |
| 見込 (prospect) | 商談につながった件数 |
| 受注 (received) | 成約件数 |
| 送客 (referral) | 他社への紹介件数 |
| 資料 (doc) | 資料送付件数 |
| 追客 (follow_up) | フォローアップ件数 |
| 稼働時間 (work_hours) | その日の勤務時間 |
| 出勤 (work_day) | 出勤フラグ |
| 生産性 (productivity) | = (見込 + 追客×0.5) ÷ 稼働時間 |
| 有効率 | = 有効 ÷ 架電 × 100 |
| 受注率 | = 受注 ÷ 架電 × 100 |
