# SalesMgr Access アプリケーション 仕様書

## 概要
営業チームの日次活動記録・目標管理・送客管理・レポート閲覧を行うAccessアプリケーション。
起動すると F_Main（メインメニュー）が表示される。

---

## オブジェクト一覧

### テーブル（4個）

| テーブル名 | 件数 | 用途 |
|-----------|------|------|
| T_MEMBERS | 11 | 担当者マスタ（名前、有効/無効） |
| T_RECORDS | 5,573 | 日次活動記録（架電・成果・時間帯別） |
| T_MEMBER_TARGETS | 12 | 月次目標設定（担当者×年月） |
| T_REFERRALS | 4,837 | 送客記録 |

### クエリ（12個）

| クエリ名 | パラメータ | 用途 |
|---------|-----------|------|
| Q_ActiveMembers | なし | 有効担当者一覧 |
| Q_Team_Monthly_Sum | prmDateFrom, prmDateTo | チーム月次集計（架電/有効/見込/受注/稼働h） |
| Q_Team_Monthly_Targets | prmYear, prmMonth | チーム月次目標合計 |
| Q_Member_Monthly_Sum | prmDateFrom, prmDateTo | 担当者別月次集計 |
| Q_Rank_Received | prmDateFrom, prmDateTo | 受注ランキング |
| Q_Rank_Prospect | prmDateFrom, prmDateTo | 見込ランキング |
| Q_Rank_Referral | prmDateFrom, prmDateTo | 送客ランキング |
| Q_Rank_Productivity | prmDateFrom, prmDateTo | 生産性ランキング |
| Q_Hourly_By_Member | prmDateFrom, prmDateTo | 時間帯別架電集計 |
| Q_Trend_Monthly | prmTrendStart, prmTrendEnd | 月別推移（6ヶ月） |
| Q_Member_12Month | prm12Start, prm12End | 担当者別12ヶ月推移 |
| Q_RefTrend_Monthly | prm12Start, prm12End | 送客月別推移 |

### フォーム（8個）

| フォーム名 | 用途 |
|-----------|------|
| F_Main | メインメニュー |
| F_Daily | 日次記録一覧 |
| F_DailyEdit | 日次記録編集（ポップアップ） |
| F_Members | 担当者管理 |
| F_Targets | 月次目標設定 |
| F_Referrals | 送客登録 |

## フォーム詳細仕様

### F_Main（メインメニュー）

| ボタン | 動作 |
|--------|------|
| 日次登録 | F_Daily を開く |
| 目標設定 | F_Targets を開く |
| 送客登録 | F_Referrals を開く |
| レポート | F_Report を開く |
| ランキング | F_Ranking を開く |
| 担当者管理 | F_Members を開く |

---

### F_Daily（日次記録一覧）

| コントロール | 種類 | 動作 |
|------------|------|------|
| ◀ | ボタン | 前月へ移動 |
| ▶ | ボタン | 翌月へ移動 |
| lblMonth | ラベル | 「YYYY年M月」表示 |
| cboMember | コンボ | 担当者フィルタ（有効メンバーのみ）。選択で一覧Requery |
| 新規登録 | ボタン | F_DailyEdit を ADDモードで開く（acDialog） |
| 編集 | ボタン | 選択行を F_DailyEdit EDITモードで開く |
| 削除 | ボタン | 確認ダイアログ後、選択行を DELETE |
| lstRecords | リスト | 当月レコード一覧（ID/日付/担当者/架電/有効/見込/資料/追客/受注/稼働h/備考） |

**ダブルクリック**: 編集と同じ動作

---

### F_DailyEdit（日次記録編集 ポップアップ）

**モード**: OpenArgs で "ADD|年|月" または "EDIT|ID" を受け取る

| コントロール | 種類 | 説明 |
|------------|------|------|
| txtRecDate | テキスト | 日付 (yyyy/mm/dd) |
| cboMember | コンボ | 担当者選択 |
| txtWorkHours | テキスト | 稼働時間（初期値: 8） |
| chkWorkDay | チェック | 出勤フラグ（初期値: True） |
| txtC10〜txtC18 | テキスト×9 | 時間帯別架電（10時〜18時、初期値: 0） |
| txtValid | テキスト | 有効件数 |
| txtProspect | テキスト | 見込件数 |
| txtDoc | テキスト | 資料件数 |
| txtFollow | テキスト | 追客件数 |
| txtReceived | テキスト | 受注件数 |
| txtReferral | テキスト | 送客件数 |
| txtNote | テキスト | 備考 |
| 保存 | ボタン | INSERT（新規）/ UPDATE（編集）実行後、フォームを閉じる |
| キャンセル | ボタン | フォームを閉じる |

**保存ロジック**: calls = txtC10〜txtC18の合計を自動計算
